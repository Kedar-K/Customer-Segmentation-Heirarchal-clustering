How to use?

This guide will help to understand the system step by step.

  1. Go to the home page and press the button get started, this action will redirect to upload page
  2. csv file can be uploaded on this page by pressing upload button and selecting the file from local machine and press submit.
  3. Instructions to upload file
    • Make sure that file name is "Mall_Customers"
    • only ".csv" extension is allowed
  4. Pressing the submit button will lead to results page where one can get the generated results
